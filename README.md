# 👗 Ameerah Clothing

A modern full-stack eCommerce web application built for a clothing brand. This project provides a seamless and responsive shopping experience with dynamic product listings, category management, and user-friendly design.

---

## 🚀 Features

* 🛍️ Browse products by categories
* 🔎 Search and filter products
* 🧾 Product details page
* 🛒 Add to cart functionality
* ⚡ Real-time updates (if using backend like Supabase)
* 📱 Fully responsive design
* 🎨 Modern UI/UX
