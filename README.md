# 👗 Ameerah Clothing

A modern full-stack eCommerce web application built for a clothing brand. This project provides a seamless and responsive shopping experience with dynamic product listings, category management, and user-friendly design.

---

## 🚀 Features

* 🛍️ Browse products by categories
* 🔎 Search and filter products
* 🧾 Product details page
* 🛒 Add to cart functionality
* ⚡ Real-time updates (if using backend like Supabase)
* 📱 Fully responsive design
* 🎨 Modern UI/UX

---

## 🛠️ Tech Stack

**Frontend:**

* React.js
* Vite
* Tailwind CSS

**Backend / Database:**

* Supabase

**Other Tools:**

* Git & GitHub
* EmailJS 

---

## 📦 Installation & Setup

1. Clone the repository:

```bash
git clone https://github.com/your-username/ameerah-clothing.git
```

2. Navigate to the project folder:

```bash
cd ameerah-clothing
```

3. Install dependencies:

```bash
npm install
```

4. Run the development server:

```bash
npm run dev
```
